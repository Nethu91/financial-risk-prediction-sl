Financial Risk Prediction using Explainable Machine Learning

Project Overview

This research develops an explainable machine learning model to predict financial risk among Sri Lankan university students.

The study compares multiple supervised machine learning algorithms and identifies the most suitable model for predicting financial vulnerability.

Objectives

- Predict financial risk
- Compare ML algorithms
- Explain predictions using SHAP
- Support early intervention

Machine Learning Models

- Logistic Regression
- Decision Tree
- Random Forest
- XGBoost

Explainability

SHAP (SHapley Additive Explanations)

Dataset

Primary Dataset collected using Google Forms.

Expected Responses-300-500 Students
